# Covid19 Scrapper

![N|Solid](https://image.flaticon.com/icons/png/512/387/387561.png)

Covid19-scrapper scrapes statistics from wikipedia.

# Features:
  - Scrapes Country flag, country name, infected, dead, recovered.
  - Converts to Json.
  - Stores in file.
### Installation/Usage
```sh
$ npm install
$ node index.js
```
